브라우저가 파일을 읽는 일반적인 방법 : parsing HTML - blocked(fetching js - excuting js) - parsing HTML

브라우저가 파일을 읽는 부수적인 방법 : parsing HTML - fetching js - excuting js(js 파일을 닫는 body 태그 앞에 배치)

브라우저가 파일을 읽는 head + async 방법 : 병렬 진행(parsing HTML - fetching js) - blocked(excuting js) - parsing HTML

브라우저가 파일을 읽는 head + defer 방법 : parsing HTML - fetching js - parsing HTML - excuting js

head + defer 방법이 가장 안전하고 효율적이다.
